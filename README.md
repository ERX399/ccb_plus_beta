# C C B PLUS Beta

基于 [Koikokokokoro/astrbot_plugin_ccb_plus](https://github.com/Koikokokokoro/astrbot_plugin_ccb_plus) 的个人定制增强版。

> Beta 分支为个人需求定制版，功能更激进；如需稳定原版，请使用原作者仓库。
## Beta 版新增内容（共 20 项）


### 权限与群聊控制
1. **群聊白名单**：可限制仅指定群聊使用插件。
2. **群聊单独配置**：使用 `template_list` 为不同群单独设置参数。
3. **群聊每日 CCB 次数限制**：每个群可单独设置每日 `/ccb` 次数上限。
4. **群聊未授权静默返回**：不再刷屏提示“未开启功能”。

### 用户保护与管理
5. **管理员默认加入白名单**：插件会默认把 AstrBot 管理员加入 `white_list`。
6. **机器人自身默认加入白名单**：机器人自身 ID 会自动加入 `white_list`。
7. **`ccbnodo` 所有人指令**：快捷切换指定用户防被 CCB 状态。
8. **`ccbnodo list` 指令**：以合并转发查看保护名单，自动排除 AstrBot 管理员和机器人。
9. **`ccbclear` 管理员指令**：清理指定用户的被 CCB 与 CCB 他人记录。

### 暴击与管理员增强
10. **管理员暴击倍率**：可配置管理员触发暴击时使用自定义倍率。
11. **暴击倍率显示一致**：管理员使用自定义倍率时仍显示普通“暴击”。
12. **管理员额外暴击率**：管理员可获得额外暴击概率。

13. **管理员豁免养胃**：管理员可配置为不触发频率养胃/随机养胃。
14. **管理员注入量保底值**：管理员发起 `/ccb` 时可配置最终注入量最低值。

### 显示与发送方式
15. **头像显示开关**：可关闭 CCB 结果中的头像图片。
16. **合并转发发送**：支持尝试用合并转发发送 CCB 结果，不支持时自动回退普通消息。
17. **TOP 榜数量配置**：所有排行榜默认显示数量可配置，命令也可追加数量，最大 100。
18. **TOP 榜自动合并转发**：排行榜超过 10 条时强制合并转发；失败时普通文字回退 TOP10。
19. **全群日志汇总榜**：新增 `-all` 后缀命令，基于完整日志汇总全部群数据。
20. **日志配置置顶**：`is_log` 放到配置最上方，方便管理。
## 当前主要指令

### 基础指令

| 指令 | 权限 | 说明 |
|---|---|---|
| `/ccb [@目标]` | 所有人 | 执行 CCB |
| `/ccbinfo [@目标]` | 所有人 | 查询统计 |

### 当前群排行榜

| 指令 | 权限 | 说明 |
|---|---|---|
| `/ccbtop [数量]` | 所有人 | 被 CCB 次数榜 |
| `/ccbvol [数量]` | 所有人 | 累计注入量榜 |
| `/ccbmax [数量]` | 所有人 | 单次最大榜 |
| `/xnn [数量]` | 所有人 | 小南梁榜 |

### 用户保护指令

| 指令 | 权限 | 说明 |
|---|---|---|
| `/ccbnodo [@目标]` | 所有人 | 切换保护 |
| `/ccbnodo list` | 所有人 | 查看保护名单 |

### 管理员指令

| 指令 | 权限 | 说明 |
|---|---|---|
| `/ccbclear [@目标]` | 管理员 | 清除记录 |
### 全群日志汇总排行榜

> 基于完整日志 `data/ccb_log.json` 汇总全部群数据。

| 指令 | 权限 | 说明 |

|---|---|---|
| `/ccbtop-all [数量]` | 所有人 | 全群次数榜 |
| `/ccbvol-all [数量]` | 所有人 | 全群注入量榜 |
| `/ccbmax-all [数量]` | 所有人 | 全群单次最大榜 |
| `/xnn-all [数量]` | 所有人 | 全群小南梁榜 |


## 配置结构亮点

- `is_log`：日志配置置顶。
- `display_settings`：显示设置折叠配置，包含头像、合并转发、TOP 榜默认数量等。
- `group_configs`：群聊单独配置，使用 `template_list`。
- `admin_settings`：管理员增强折叠配置，包含暴击倍率、额外暴击率、养胃豁免、注入量保底等。

## 数据文件

- 主数据：`data/ccb.json`
- 完整日志：`data/ccb_log.json`
- 群聊每日次数统计：`data/ccb_daily_limit.json`
